# Herramientas de SEO para el Proyecto

## Herramientas Gratuitas o de Bajo Coste

### 1. Google Keyword Planner
- **Propósito**: Investigación de keywords gratuita
- **Funcionalidades**:
  - Volumen de búsquedas
  - Competencia
  - Sugerencias de keywords
  - Tendencias temporales
- **Integración**: Parte de Google Ads
- **Coste**: Gratuito con cuenta de Google Ads

### 2. Google Search Console
- **Propósito**: Monitoreo de rendimiento en Google
- **Funcionalidades**:
  - Rendimiento en búsquedas
  - Errores de indexación
  - Enlaces externos
  - Experiencia de usuario móvil
- **Integración**: API oficial de Google
- **Coste**: Gratuito

### 3. Google Analytics
- **Propósito**: Análisis de tráfico y comportamiento
- **Funcionalidades**:
  - Flujo de usuarios
  - Comportamiento en sitio
  - Conversiones
  - Datos demográficos
- **Integración**: API oficial de Google
- **Coste**: Gratuito

### 4. Screaming Frog
- **Propósito**: Auditoría técnica del sitio
- **Funcionalidades**:
  - Análisis de estructura
  - Detección de errores
  - Validación de meta tags
  - Análisis de contenido
- **Integración**: Exportación de datos
- **Coste**: Versión gratuita limitada (500 URLs)

### 5. UberSuggest
- **Propósito**: Investigación de keywords y análisis de competencia
- **Funcionalidades**:
  - Sugerencias de keywords
  - Análisis de competencia
  - Seguimiento de posiciones
  - Auditorías de sitio
- **Integración**: API disponible
- **Coste**: Plan gratuito limitado, planes desde 29€/mes
- **Nota**: Si ya tienes cuenta, podemos explorar su integración con Cursor

### 6. AnswerThePublic
- **Propósito**: Descubrimiento de preguntas relacionadas
- **Funcionalidades**:
  - Visualización de preguntas
  - Análisis de intención de búsqueda
  - Sugerencias de contenido
- **Integración**: API disponible
- **Coste**: Versión gratuita limitada

### 7. Grammarly
- **Propósito**: Corrección y mejora de texto
- **Funcionalidades**:
  - Corrección gramatical
  - Mejora de estilo
  - Tono de voz
- **Integración**: API disponible
- **Coste**: Versión gratuita

### 8. Ubersuggest Chrome Extension
- **Propósito**: Análisis rápido de keywords y competencia
- **Funcionalidades**:
  - Análisis de keywords en tiempo real
  - Sugerencias de contenido
  - Análisis de competencia
- **Integración**: Extensión de Chrome
- **Coste**: Gratuito con limitaciones

## Herramientas de Pago (para considerar en el futuro)

- **Funcionalidades**: Análisis completo
- **Coste**: Desde 99€/mes

### 2. Ahrefs
- **Propósito**: Análisis de backlinks y keywords
- **Funcionalidades**: Análisis completo
- **Coste**: Desde 99€/mes

### 3. Clearscope
- **Propósito**: Optimización de contenido
- **Funcionalidades**: Análisis completo

## Integración con Cursor

Para integrar UberSuggest con Cursor, podemos explorar las siguientes opciones:

1. **API de UberSuggest**: Si UberSuggest ofrece una API, podemos crear un módulo en nuestro proyecto que se conecte a ella para obtener datos de keywords y análisis de competencia.

2. **Exportación de datos**: Podemos exportar datos desde UberSuggest y crear scripts para procesarlos e incorporarlos a nuestro proyecto.

3. **Extensión de Chrome**: Podemos utilizar la extensión de Chrome de UberSuggest para análisis rápidos y luego incorporar manualmente los insights a nuestro proyecto.

4. **Integración con Google Sheets**: Si UberSuggest permite exportar a Google Sheets, podemos crear una integración entre Google Sheets y nuestro proyecto para automatizar la actualización de datos.
